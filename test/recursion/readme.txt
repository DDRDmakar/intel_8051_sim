 
Запуск программы:
-dv --mode bin -i recursion.bin -o snapshots/dump.json -b ^0

Результат работы программы сохраняется в папку snapshots.

Программа рекурсивно вызывает функцию, тем самым итерируясь по массиву чисел: 0, 1, 2 и т.д.
Когда число в текущей ячейке массива оказывается равно искомому, то
оно записывается в порт P0, а программа завершается.

Результат должен быть равен 228
(шестнадцатеричное #E4), а массив должен быть 
расположен в памяти данных, как показано на скриншоте  screenshot.png.

"P0": "#E4"
