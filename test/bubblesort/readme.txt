
Запуск программы:
'../../build/sim8051' -dv --mode text -i sort.json -o snapshots/dump.json

В результате в папке snapshots должен появиться файл dump.json. В поле "data" (память данных) должен располагаться отсортированный массив чисел от 0 до 15, как это показано на скриншоте (файл screenshot.png).
