 
Запуск программы:
'../../build/sim8051' -dv --mode text -i loop.json -o snapshots/dump.json

Результат работы программы сохраняется в папку snapshots.

Программа берёт число 7, в цикле 4 раза умножает его на 2,
а затем в другом цикле 22 раза прибавляет к нему 3.
Результат записывается в порт P0.

Результат должен быть равен (7 * 2^4) + (3 * 22) = 7 * 16 + 66 = 178
(шестнадцатеричное #B2)

Значит после окончания работы программы порт P0 должен быть равен #B2

"P0": "#B2"
